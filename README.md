# ⛏️ Minecraft OP Blocks Datapack

Welcome to the official repository for the **Minecraft, but every block is OP** datapack! This project completely changes your survival experience by dropping end-game items from basic blocks.

---

## 🚀 Features

*   **💎 Diamond Rain:** Breaking a single `dirt` block instantly drops a full stack (**64x Block of Diamonds**).
*   **🪙 Hourly Netherite:** Once every hour (every 72,000 game ticks), breaking a block will also reward you with **Netherite Ingots** and a **Smithing Table** for your upgrades.

---

## 🛠️ How to Install

1.  Click the green **Code** button at the top of this page and select **Download ZIP**.
2.  Open Minecraft and go to your **Singleplayer** world list.
3.  Select your world, click **Edit**, and then click **Open World Folder**.
4.  Open the folder named `datapacks`.
5.  Drop the downloaded `.zip` file inside that folder.
6.  Join your world (or type `/reload` in the chat if you are already inside).
